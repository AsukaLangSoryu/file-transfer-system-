# 测试报告

## 1. 测试环境

- 操作系统: Windows/Linux/macOS
- Python版本: 3.8+
- 网络环境: 本地回环/局域网

## 2. 测试场景

### 2.1 基础功能测试

#### 测试1: 三次握手
**目的**: 验证连接建立过程
**步骤**:
1. 启动服务器
2. 客户端发起连接
3. 观察握手过程

**预期结果**: 成功建立连接

#### 测试2: 数据传输
**目的**: 验证可靠数据传输
**步骤**:
1. 建立连接
2. 传输测试文件
3. 验证接收数据完整性

**预期结果**: 数据完整无误

#### 测试3: 四次挥手
**目的**: 验证连接断开过程
**步骤**:
1. 传输完成后断开连接
2. 观察挥手过程

**预期结果**: 正常断开连接

### 2.2 可靠性测试

#### 测试4: 丢包重传
**目的**: 验证超时重传机制
**步骤**:
1. 模拟10%丢包率
2. 传输数据
3. 检查重传次数

**预期结果**: 自动重传丢失数据包

#### 测试5: 乱序处理
**目的**: 验证乱序数据包处理
**步骤**:
1. 模拟数据包乱序
2. 传输数据
3. 验证接收顺序

**预期结果**: 正确重组数据

### 2.3 性能测试

#### 测试6: 吞吐量测试
**目的**: 测试传输速率
**步骤**:
1. 传输大文件（10MB）
2. 记录传输时间
3. 计算吞吐量

**预期结果**: 达到合理的传输速率

#### 测试7: 拥塞控制测试
**目的**: 验证拥塞控制算法
**步骤**:
1. 模拟网络拥塞
2. 观察cwnd变化
3. 验证慢启动和拥塞避免

**预期结果**: cwnd正确调整
