# 测试文件说明

## 文件列表

### 1. 文本文件测试
- `test_text.txt` - 基础文本文件（约500字节）
- `large_text.txt` - 大文本文件（用于测试多线程传输）

### 2. 图片文件测试
需要准备：
- `test_image_small.jpg` - 小图片（< 1MB）
- `test_image_large.jpg` - 大图片（> 5MB，测试多线程）

### 3. 视频文件测试
需要准备：
- `test_video.mp4` - 测试视频文件（建议10-50MB）

## 测试步骤

### 功能1：单文件上传
1. 打开传输客户端页面
2. 拖放或选择 test_text.txt
3. 观察上传进度
4. 验证文件出现在文件库中

### 功能2：多线程并发传输（≥3个）
1. 同时选择3个或更多文件
2. 观察是否有3个传输任务同时进行
3. 检查每个任务的独立进度条

### 功能3：文件预览
- 文本：点击文件应显示文本内容
- 图片：点击应显示图片预览
- 视频：点击应能在线播放

### 功能4：群发功能
1. 进入群发广播页面
2. 选择多个目标设备
3. 选择文件
4. 执行广播
5. 验证所有目标设备都收到文件
